帮助文档

系统使用说明与常见问题

🚀 快速开始

欢迎使用 AI Scanner 管理后台。以下是主要功能模块说明：

  • 仪表盘：查看系统核心数据，包括用户数、扫描次数、兑换码状态等。
  • 资金管理：查看所有支付订单，统计 USDT 收入情况。
  • 应用管理：管理已上架应用，支持搜索、筛选、上下架操作。
  • 用户管理：查看注册用户列表，调整用户 Credits 余额。
  • 兑换码：批量生成兑换码，查看使用记录，撤销无效码。
  • 数据统计：查看应用下载、评论、喜爱的汇总统计。
  • 账号设置：修改当前管理员密码，管理其他管理员账号。

🎫 兑换码说明

  • 兑换码格式：XXXX-XXXX-XXXX（12位，使用无歧义字符）
  • 一次性码：每码可用次数设为 1，一人兑换后失效。
  • 共享码：每码可用次数 >1，可多人使用同一码。
  • 有效期：设为 0 表示永不过期；否则为从生成起的天数。
  • 已撤销的兑换码无法再被使用，操作不可逆。

💳 支付说明

  • 当前支持的支付方式：USDT (TRC20) via Cryptomus。
  • 订单状态：paid = 已完成，pending = 待付款，expired = 已过期。
  • 支付成功后系统自动为用户增加对应 Credits。

⚙️ 系统维护

  • 科技资讯每天凌晨 3:00 自动从 RSS 源拉取并存入数据库。
  • 超过 30 天的资讯会自动清理。
  • 如需手动触发：在服务器执行 docker exec ai-scanner-backend-1 python3 /app/fetch_news.py

❓ 常见问题

  1. 用户反馈兑换码无效？
    在"兑换码管理"中搜索该码，检查是否已过期或被撤销，查看使用次数是否已满。
  2. 如何给用户补偿 Credits？
    在"用户管理"中找到该用户，点击"调整 Credits"，输入正数进行补充。
  3. 如何下架某个违规应用？
    在"应用管理"中找到该应用，点击"下架"按钮即可从市场隐藏。
  4. 忘记管理员密码？
    联系超级管理员通过数据库重置，或使用 API 端点直接操作。
